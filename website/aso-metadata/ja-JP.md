## Description (4000 chars max)
```
延々とスクロールするのはもう終わり。FastFlixに見たい気分を伝えるだけで、AIが数秒で最適な映画やドラマを見つけます — あなたの配信サービスで視聴可能なものだけ。

使い方
見たいものを自然な言葉で説明：ジャンル、俳優、気分、時代。AIがあなたを理解し、評価順にパーソナライズされたおすすめを提案します。

学習するおすすめ機能
• 視聴した映画やドラマを評価（1〜5つ星）
• あなたの好みプロフィールが時間とともに精度向上
• 毎日のおすすめとトレンドをあなたの好みに基づいてソート
• AI搭載の「あなたへ」セクション

配信サービスでフィルタリング
• Netflix、Disney+、Prime Video、Apple TV+など20以上のサービスから視聴可能なコンテンツのみ表示
• サブスクリプション、レンタル、購入の可否が即座にわかる
• 今週の新着をあなたの好みでフィルタリング

あなたのウォッチリスト
• 見たい映画やドラマを保存
• 視聴済みを記録
• おすすめを見逃さない

決定に必要な情報が全て
• 評価、あらすじ、キャスト、監督、ジャンル
• 国別ストリーミング配信状況
• 6言語対応：日本語、英語、フランス語、ドイツ語、スペイン語、イタリア語

何を見るか迷ったら？
検索ボタンをタップしてAIにお任せ。「長い一日の後に軽いもの」や「ハラハラするスリラー」など気分を伝えるだけ — FastFlixが理解します。

サブスクリプション

• 月額：€1.99/月
• 3ヶ月：€3.99/3ヶ月
• 年額：€9.99/年
• 確認時にApple IDに請求
• 期間終了24時間前までにキャンセルしない限り自動更新
• 設定 > Apple ID > サブスクリプションで管理

FastFlixをダウンロードして、もう選ぶ時間を無駄にしない。
```
